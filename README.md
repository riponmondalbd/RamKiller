# Ram Killer Pro

**Ram Killer Pro** is a powerful Chrome extension designed to optimize your browser's performance and save system resources. It automatically suspends inactive tabs, freeing up valuable RAM and improving overall system speed.

## ✨ Features

- **Auto Suspend Idle Tabs**: Automatically suspends tabs after a user-defined period of inactivity.
- **Multiple Suspension Mode Options**:
  - **Discard**: Maximum memory saving by fully discarding the tab.
  - **Auto Reload**: Automatically reloads the tab when you switch back to it.
  - **Manual Reload**: Requires a click to reload the suspended tab.
- **Smart Exceptions**:
  - Automatically skips pinned tabs.
  - Automatically skips tabs currently playing audio/video.
  - Custom Domain Whitelist (supports wildcards e.g., `*.youtube.com`, `github.com`).
- **Real-Time Statistics**: Track suspended tabs, total open tabs, and estimated memory recovered directly from the popup.
- **One-Click Kill**: An emergency "Kill Idle Tabs Now" button in the popup menu to instantly clear memory.
- **Modern User Interface**: A beautifully designed, easy-to-use popup and settings page with a professional and clean aesthetic.

## 🚀 Installation (Local / Developer Mode)

To install and test this extension on your browser via developer mode:

1. Clone or download this repository to your local machine.
2. Open Google Chrome/Brave/Edge and navigate to `chrome://extensions/`.
3. Enable **"Developer mode"** in the top right corner.
4. Click on **"Load unpacked"**.
5. Select the `RamKiller` folder where this repository was cloned/downloaded.
6. The *Ram Killer Pro* icon should now appear in your browser's toolbar! Pin it for quick access.

## 🛠️ Usage & Configuration

- **Popup Menu**: Click the extension icon to view RAM recovery statistics and access quick actions.
- **Settings Page**: Click the Settings gear icon in the popup, or right-click the extension icon and select "Options." From there, you can configure your idle timeout limits, manage your suspension mode, and add custom domain exceptions.
